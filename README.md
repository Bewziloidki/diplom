# Telegram Bot: Social Media Status Monitor

## Описание проекта
Данный проект представляет собой Telegram-бота, разработанного на языке Python,  
предназначенного для мониторинга доступности социальных сетей и онлайн-сервисов.

Бот позволяет пользователю:
- проверять текущий статус популярных сервисов;
- получать информацию о времени последней проверки;
- просматривать историю сбоев;
- настраивать уведомления о недоступности сервисов.

Проект создан в рамках итоговой (дипломной) работы и демонстрирует навыки работы  
с Telegram Bot API, асинхронным программированием, внешними API и хранением данных.



## Функциональные возможности

Основные команды бота:

- `/start` — запуск бота и регистрация пользователя  
- `/help` — список доступных команд  
- `/status` — текущий статус отслеживаемых сервисов  
- `/add` — добавить сервис для мониторинга  
- `/remove` — удалить сервис из списка  
- `/history` — история сбоев и проверок  
- `/settings` — настройки уведомлений
